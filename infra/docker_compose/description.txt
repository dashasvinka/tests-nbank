Все контейнеры, которые необходимо запускать для прогона:
- фронтенд // 80
- бекенд // 4111 -> 4111
- nginx // 80 -> 80
- selenoid // 4444 -> 4444
- selenoid-ui // 80 -> 80

Управление Docker Compose:
- создание docker-compose.yaml
- описать каждый докер контейнер (образ, порт, последовательность, общая докер сеть)
- скрипт по поднятию всех докер контейнеров с помощью Docker Compose
